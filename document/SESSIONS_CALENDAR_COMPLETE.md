## 🔌 الاتصال بـ API

### Endpoints المستخدمة:

```javascript
1. GET /api/sessions/
   - تحميل جميع الحصص
   - يتم استدعاؤه عند تحميل الصفحة
   - يتم استدعاؤه بعد كل عملية (إضافة/تعديل/حذف)

2. GET /api/teachers/
   - تحميل قائمة المحفظين
   - لملء Dropdown الفلترة
   - لملء Dropdown النموذج

3. GET /api/students/
   - تحميل قائمة الطلبة
   - لملء Dropdown الفلترة
   - لملء Dropdown النموذج

4. POST /api/sessions/
   - إضافة حصة جديدة
   - يرسل جميع بيانات الحصة
   - يعيد بيانات الحصة المضافة

5. PUT /api/sessions/{id}/
   - تعديل حصة موجودة
   - يرسل جميع بيانات الحصة
   - يعيد بيانات الحصة المحدثة

6. DELETE /api/sessions/{id}/
   - حذف حصة
   - يحذف الحصة نهائياً
   - يعيد 204 No Content
```

### مثال على البيانات المرسلة:

```json
{
  "student_id": 1,
  "teacher_id": 1,
  "session_date": "2024-01-15",
  "session_time": "10:00",
  "duration_minutes": 60,
  "status": "scheduled",
  "is_makeup": false,
  "notes": "حصة عادية"
}
```

## 🎯 كيفية الاستخدام

### 1. فتح الصفحة
```
http://localhost:8080/pages/sessions.html
```

### 2. عرض الحصص
- الحصص تظهر تلقائياً في التقويم
- كل حصة لها لون حسب حالتها
- العنوان: "اسم الطالب - اسم المحفظ"

### 3. إضافة حصة جديدة
**طريقة 1:** اضغط على زر "إضافة حصة" في الأعلى
**طريقة 2:** اضغط على أي تاريخ في التقويم (التاريخ يُملأ تلقائياً)

### 4. عرض تفاصيل حصة
- اضغط على أي حصة في التقويم
- ستظهر نافذة بجميع التفاصيل:
  - الطالب
  - المحفظ
  - التاريخ والوقت
  - المدة
  - الحالة
  - النوع
  - الملاحظات

### 5. تعديل حصة
- افتح تفاصيل الحصة
- اضغط على زر "تعديل"
- عدل البيانات
- احفظ

### 6. حذف حصة
- افتح تفاصيل الحصة
- اضغط على زر "حذف"
- أكد الحذف

### 7. الفلترة
- اختر محفظ معين → تظهر حصصه فقط
- اختر طالب معين → تظهر حصصه فقط
- اختر حالة معينة → تظهر الحصص بهذه الحالة
- يمكن الجمع بين الفلاتر
- اضغط "إعادة تعيين" لإلغاء الفلاتر
