# ميزة الجدول الأسبوعي والحصص

## نظرة عامة
تم إضافة نظام متكامل لإدارة الجدول الأسبوعي والحصص مع ربط تلقائي بين الحصص المجدولة والحصص الفعلية.

## الميزات المضافة

### 1. صفحة الجدول الأسبوعي (`/scheduled-sessions/`)

#### إضافة حصة للجدول
- نموذج شامل لإضافة حصة مجدولة جديدة
- اختيار الطالب والمحفظ
- تحديد اليوم والوقت والمدة
- خيار لإنشاء حصة تلقائياً في صفحة الحصص
- عند تفعيل خيار "إنشاء حصة تلقائياً"، يتم:
  - حساب التاريخ القادم لليوم المحدد
  - إنشاء حصة في صفحة الحصص بحالة "مجدولة"
  - ربط الحصة بالحصة المجدولة

#### عرض تفاصيل الحصة المجدولة
- عرض جميع معلومات الحصة المجدولة
- اسم الطالب والمحفظ
- اليوم والوقت والمدة
- الحالة (نشط/غير نشط)

#### تعديل الحصة المجدولة
- تعديل جميع بيانات الحصة المجدولة
- تغيير الطالب أو المحفظ
- تعديل اليوم والوقت والمدة
- تفعيل أو تعطيل الحصة

#### حذف الحصة المجدولة
- حذف الحصة المجدولة مع تأكيد
- تحديث تلقائي للجدول والإحصائيات

#### إنشاء حصص الأسبوع
- زر جديد "إنشاء حصص الأسبوع"
- ينشئ تلقائياً حصص للأسبوع القادم لجميع الحصص المجدولة النشطة
- يحسب التواريخ بناءً على أيام الأسبوع
- يعرض عدد الحصص المنشأة والأخطاء إن وجدت

### 2. صفحة الحصص (`/sessions/`)

#### إضافة حصة جديدة
- نموذج شامل لإضافة حصة فعلية
- اختيار الطالب والمحفظ
- تحديد التاريخ والوقت
- اختيار الحالة (مجدولة، حضر، غائب، معتذر، ملغية)
- خيار لتحديد إذا كانت حصة تعويضية

#### عرض تفاصيل الحصة
- الانتقال لصفحة تفاصيل الحصة الكاملة

#### تعديل الحصة
- تعديل جميع بيانات الحصة
- تغيير الطالب أو المحفظ
- تعديل التاريخ والوقت
- تغيير الحالة
- تحديد إذا كانت تعويضية

#### حذف الحصة
- حذف الحصة مع تأكيد
- تحديث تلقائي للجدول والإحصائيات

## كيفية الاستخدام

### إضافة حصة للجدول الأسبوعي
1. افتح صفحة الجدول الأسبوعي
2. اضغط على "إضافة حصة للجدول"
3. املأ البيانات المطلوبة
4. فعّل خيار "إنشاء حصة تلقائياً" إذا أردت إنشاء حصة فعلية
5. اضغط "إضافة"

### إنشاء حصص الأسبوع القادم
1. افتح صفحة الجدول الأسبوعي
2. اضغط على "إنشاء حصص الأسبوع"
3. أكد العملية
4. سيتم إنشاء حصص لجميع الحصص المجدولة النشطة

### إضافة حصة فردية
1. افتح صفحة الحصص
2. اضغط على "إضافة حصة جديدة"
3. املأ البيانات المطلوبة
4. اضغط "إضافة"

## الربط بين الحصص المجدولة والحصص الفعلية

- كل حصة فعلية يمكن أن تكون مرتبطة بحصة مجدولة عبر حقل `scheduled_session`
- عند إنشاء حصة من الجدول الأسبوعي، يتم ربطها تلقائياً
- يمكن تتبع جميع الحصص الفعلية المرتبطة بحصة مجدولة معينة

## الإحصائيات

### صفحة الجدول الأسبوعي
- إجمالي الحصص المجدولة
- الحصص النشطة
- الحصص غير النشطة
- عدد الطلبة في الجدول

### صفحة الحصص
- إجمالي الحصص
- الحصص المكتملة (حضر)
- الحصص المجدولة
- الحصص الملغية

## الملفات المعدلة

1. `frontend/pages/scheduled-sessions.html` - إضافة زر إنشاء حصص الأسبوع
2. `frontend/js/scheduled-sessions.js` - إضافة جميع الوظائف (إضافة، عرض، تعديل، حذف، إنشاء أسبوعي)
3. `frontend/pages/sessions.html` - بدون تغيير
4. `frontend/js/sessions.js` - إضافة جميع الوظائف (إضافة، تعديل، حذف)

## ملاحظات تقنية

- جميع العمليات تستخدم API الخاص بالنظام
- التحقق من البيانات يتم على مستوى الواجهة والخادم
- رسائل الخطأ واضحة ومفصلة
- تحديث تلقائي للجداول والإحصائيات بعد كل عملية
- استخدام SweetAlert2 لواجهة مستخدم جميلة وسهلة

## التطوير المستقبلي

- إضافة إمكانية نسخ حصة مجدولة
- إضافة إمكانية تعطيل حصة مجدولة مؤقتاً
- إضافة تقرير للحصص المجدولة والفعلية
- إضافة إشعارات للحصص القادمة
- إضافة إمكانية إنشاء حصص لفترة محددة (شهر، ربع سنة، إلخ)
