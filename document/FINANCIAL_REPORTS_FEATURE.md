# ميزة التقارير المالية الشاملة

## نظرة عامة
تم إضافة تبويب جديد في صفحة التقارير يعرض ملخص مالي شامل لأي شهر محدد، يشمل جميع الإيرادات والمصروفات من جميع مصادر النظام.

## المميزات

### 1. اختيار الشهر
- اختيار أي شهر وسنة
- افتراضياً يعرض الشهر الحالي
- تحديث فوري عند تغيير الشهر

### 2. الإيرادات (Income)
يتم حساب الإيرادات من مصدرين:

#### أ. دفعات الطلاب
- جميع المدفوعات المكتملة من الطلاب
- من جدول `payments`
- الحالة: `completed`

#### ب. إيداعات الخزنة
- جميع عمليات الإيداع في الخزائن
- من جدول `treasury_transactions`
- النوع: `deposit`
- المبلغ: موجب (+)

**إجمالي الإيرادات = دفعات الطلاب + إيداعات الخزنة**

### 3. المصروفات (Expenses)
يتم حساب المصروفات من مصدرين:

#### أ. رواتب المحفظين
- جميع الرواتب المدفوعة
- من جدول `teacher_salaries`
- الحالة: `paid`
- للشهر المحدد

#### ب. صرف من الخزنة
- جميع عمليات الصرف من الخزائن
- من جدول `treasury_transactions`
- النوع: `withdrawal`
- المبلغ: سالب (-)
- مصنفة حسب البند (كهرباء، إيجار، إلخ)

**إجمالي المصروفات = رواتب المحفظين + صرف من الخزنة**

### 4. صافي الربح (Net Profit)
```
صافي الربح = إجمالي الإيرادات - إجمالي المصروفات
```

- يظهر باللون الأخضر إذا كان موجب (ربح)
- يظهر باللون الأحمر إذا كان سالب (خسارة)

### 5. التفاصيل المرئية

#### بطاقات الإحصائيات
- بطاقة الإيرادات (خضراء)
- بطاقة المصروفات (ذهبية)
- بطاقة صافي الربح (بنية)

#### رسم بياني
- رسم بياني عمودي (Bar Chart)
- يقارن بين:
  - دفعات الطلاب
  - إيداعات الخزنة
  - رواتب المحفظين
  - صرف من الخزنة

#### جداول التفاصيل
- جدول تفاصيل الإيرادات
- جدول تفاصيل المصروفات (مصنف حسب البند)

## كيفية الاستخدام

### الخطوات:

1. **افتح صفحة التقارير**
   ```
   http://localhost:8000/pages/reports.html
   ```

2. **اضغط على تبويب "التقارير المالية"**
   - التبويب الثاني في القائمة

3. **اختر الشهر**
   - افتراضياً: الشهر الحالي
   - يمكن اختيار أي شهر سابق أو قادم

4. **اضغط "عرض التقرير"**
   - يتم تحميل البيانات وعرض التقرير

5. **راجع النتائج**
   - الإحصائيات في الأعلى
   - الرسم البياني في الوسط
   - التفاصيل في الأسفل

## أمثلة عملية

### مثال 1: شهر مربح
```
الشهر: مارس 2026

الإيرادات:
- دفعات الطلاب: 50,000 ج.م (100 دفعة)
- إيداعات الخزنة: 10,000 ج.م
= إجمالي الإيرادات: 60,000 ج.م

المصروفات:
- رواتب المحفظين: 30,000 ج.م (15 محفظ)
- كهرباء: 2,000 ج.م
- إيجار: 5,000 ج.م
- مصروفات إدارية: 3,000 ج.م
= إجمالي المصروفات: 40,000 ج.م

صافي الربح: 20,000 ج.م ✅ (أخضر)
```

### مثال 2: شهر بخسارة
```
الشهر: فبراير 2026

الإيرادات:
- دفعات الطلاب: 25,000 ج.م (50 دفعة)
- إيداعات الخزنة: 5,000 ج.م
= إجمالي الإيرادات: 30,000 ج.م

المصروفات:
- رواتب المحفظين: 28,000 ج.م (15 محفظ)
- كهرباء: 2,500 ج.م
- إيجار: 5,000 ج.م
- صيانة: 3,000 ج.م
= إجمالي المصروفات: 38,500 ج.م

صافي الربح: -8,500 ج.م ❌ (أحمر)
```

## مصادر البيانات

### الجداول المستخدمة:

1. **payments** - دفعات الطلاب
   ```sql
   SELECT amount, payment_date, currency_code
   FROM payments
   WHERE status = 'completed'
   AND payment_date BETWEEN 'first_day' AND 'last_day'
   ```

2. **treasury_transactions** - معاملات الخزنة
   ```sql
   SELECT amount, transaction_type, category, transaction_date, currency_code
   FROM treasury_transactions
   WHERE transaction_date BETWEEN 'first_day' AND 'last_day'
   ```

3. **teacher_salaries** - رواتب المحفظين
   ```sql
   SELECT total_amount, month, year, status, currency_code
   FROM teacher_salaries
   WHERE month = selected_month
   AND year = selected_year
   AND status = 'paid'
   ```
