# ميزة الدفع والتعديل المدمجة للفواتير

## 📋 الوصف

تم دمج زر "تعديل" و "تسجيل دفعة" في زر واحد يسمح بـ:
- ✅ تعديل تفاصيل الفاتورة (المبلغ الأساسي، الخصم، الرسوم الإضافية)
- ✅ تسجيل دفعة في نفس الوقت
- ✅ أو التعديل فقط بدون دفع (بترك المبلغ = 0)

---

## 🎯 المميزات

### 1. تعديل الفاتورة
- **المبلغ الأساسي**: يمكن تغييره
- **نسبة الخصم (%)**: يتم حساب مبلغ الخصم تلقائياً
- **مبلغ الخصم**: يُحسب تلقائياً بناءً على النسبة
- **رسوم إضافية**: يمكن إضافتها
- **سبب الخصم**: حقل اختياري
- **الإجمالي**: يُحسب تلقائياً

### 2. تسجيل الدفعة
- **المبلغ المدفوع**: يمكن إدخال أي مبلغ (أو 0 للتعديل فقط)
- **طريقة الدفع**: نقدي، تحويل بنكي، بطاقة، إلخ
- **تاريخ الدفع**: تاريخ الدفع الفعلي
- **ملاحظات**: حقل اختياري

### 3. الحسابات التلقائية
- ✅ حساب مبلغ الخصم تلقائياً عند تغيير النسبة
- ✅ حساب الإجمالي الجديد تلقائياً
- ✅ عرض المبلغ المتبقي الحالي
- ✅ التحقق من أن المبلغ المدفوع لا يتجاوز المتبقي

## 🔧 كيفية الاستخدام

### الخطوات:

1. **افتح صفحة الفواتير**
   ```
   http://localhost:8080/pages/invoices.html
   ```

2. **اضغط على زر "دفع وتعديل"** (الزر الأخضر مع أيقونتين)

3. **قسم تعديل الفاتورة** (خلفية صفراء):
   - عدّل المبلغ الأساسي
   - عدّل نسبة الخصم (سيُحسب المبلغ تلقائياً)
   - أضف رسوم إضافية إن وجدت
   - أضف سبب الخصم (اختياري)

4. **قسم تسجيل الدفعة** (خلفية خضراء):
   - أدخل المبلغ المدفوع (أو اترك 0 للتعديل فقط)
   - اختر طريقة الدفع
   - حدد تاريخ الدفع
   - أضف ملاحظات (اختياري)

5. **اضغط "حفظ التعديلات والدفعة"**

## 💡 حالات الاستخدام

### الحالة 1: تعديل وتسجيل دفعة معاً
```
المبلغ الأساسي: 1000 ج.م
نسبة الخصم: 10%
الإجمالي الجديد: 900 ج.م
المبلغ المدفوع: 500 ج.م
النتيجة: ✅ تم تحديث الفاتورة + تسجيل دفعة 500 ج.م
```

### الحالة 2: تعديل فقط بدون دفع
```
المبلغ الأساسي: 1000 ج.م
نسبة الخصم: 15%
الإجمالي الجديد: 850 ج.م
المبلغ المدفوع: 0 ج.م
النتيجة: ✅ تم تحديث الفاتورة فقط
```

### الحالة 3: دفع كامل
```
المبلغ الأساسي: 1000 ج.م
نسبة الخصم: 0%
الإجمالي: 1000 ج.م
المبلغ المدفوع: 1000 ج.م
النتيجة: ✅ تم تحديث الفاتورة + تسجيل دفعة + تغيير الحالة إلى "مدفوعة"
```
